# Enterprise Network Design using Cisco Packet Tracer

## Overview

This project presents the design and implementation of an enterprise network for a multi-site company using Cisco Packet Tracer.

The network provides secure communication between headquarters and remote branches while ensuring scalability, high availability and network segmentation.

---

## Objectives

- Design a scalable enterprise network.
- Segment departments using VLANs.
- Provide inter-VLAN communication.
- Connect remote branches using OSPF.
- Ensure high availability using HSRP.
- Provide automatic IP allocation with redundant DHCP servers.
- Publish internal and external web services.
- Restrict access using ACLs.
- Simulate Internet access through Static NAT.
